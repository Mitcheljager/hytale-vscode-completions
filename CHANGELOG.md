# Change Log

All notable changes to the "hytale-completions" extension will be documented in this file.

Follow [Keep a Changelog](http://keepachangelog.com/)

## [1.4.0]

### Added

- Completions will now complete fully when typed from nothing, completing with a default value related to their type

## [1.3.0]

### Added

- Completions for arrays will now show the type of items in the array
- Completions will now exclude some very specific values like specific numbers and colors

## [1.2.1]

### Fixes

- Fixed an error that occurred when working in a file without schema

## [1.2.0]

### Added

- JSON files for NPC/Roles will now include completions
- Arrays will now include completions

## [1.1.0]

- Added completions for Item json files
- Fixed missing completions for many keys

## [1.0.0]

- Initial release
